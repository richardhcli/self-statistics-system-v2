# Knowledge Graph

The knowledge graph is a visual map of how your concepts, themes, and entries connect over time.

## What You See

- **Nodes** represent concepts, tags, or recurring themes extracted from journal entries.
- **Edges** represent relationships between nodes (co-occurrence, causal links, progressions).
- The graph grows with each entry — new nodes and edges appear as the AI identifies connections.

## Interacting with the Graph

- Navigate to the **Graph** tab to view the full graph.
- Click a node to see related entries and stats.
- Zoom, pan, and drag to explore clusters.

## Use Cases

- Spot which topics dominate your recent entries.
- Find unexpected connections between themes.
- Track how a concept evolves across weeks or months.
